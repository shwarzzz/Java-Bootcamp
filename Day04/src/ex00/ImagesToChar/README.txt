Для запуска программы нужно:
0. Создать папку, где будут располагаться class файлы:
    mkdir target
1. Скомпилировать байт-код. Для этого нужно воспользоваться командой:
    javac -d target/ src/java/edu/school21/printer/*/*.java
Где:
     -d <директория> — флаг, который указывает, куда расположить скомпилированные классы;
2. Для запуска же необходимо использовать команду:
    java -classpath ./target/ edu.school21.printer.app.Program <путь_до_bmp> <white_color> <black_color>
Где:
     -classpath — флаг, после которого следует указать местоположение скомпилированных классов. Java будет искать главный
     класс и все сопутствующие именно в этой директории (вообще classpath указываем, только когда наш проект содержит
     много классов);
     /target — название папки, в которой находятся скомпилированные классы;
     edu.school21.printer.app.Program — указываем класс, в котором находится метод main;
     <путь_до_bmp> — путь до файл, который хотим конвертировать;
     <white_color> — символ, который заменит белый цвет;
     <black_color> — символ, который заменит черный цвет;
